# Параметры по умолчанию
```cpp
class Point{
public:
	Point(int lat = 100, int longit=100);//в месте определения, аругменты по умолчанию не пишутся
	//Этот конструктор эквивалентен конструктору по умолчанию, т.е. если мы определим конструктор по умолчанию(без параметров), компилятор выдаст ошибку так как не будет понимать какой из них вызвать
};

Point p;
Point pp{20};
Point ppp{20,20};
```
