| Escapse sequence | Значение                                    |
| ---------------- | ------------------------------------------- |
| \n               | переход курсора на новую строку             |
| \r               | перемещает курсора на начало текущей строки |
| \t               | tab                                         |
| \\               | обратный слеш                               |
| \"               | кавычка                                     |
