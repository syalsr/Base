Может ли конструктор быть шаблоном функции
?
```cpp
class Base
{
public:
    template<typename T>
    Base(T a) 
    {
        std::cout << a;
    }
};

int main()
{
    Base b{ 2 };//OK
}
```

Может ли виртуальная функция быть шаблонной?
?
Нет
```cpp
class Base
{
public:
    template<typename T>
    virtual void virt(T a) //'void Base::virt(T)' : member function templates cannot be virtual

    {
        std::cout << a;
    }
};
```
